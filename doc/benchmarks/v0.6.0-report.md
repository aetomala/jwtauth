## Notable Operations

### `BenchmarkRefreshAccessToken` — key behavioral change in v0.6.0

`RefreshAccessToken` now revokes the old refresh token on rotation (fix #195). Raw numbers:

| Run | v0.5.0 ns/op | v0.6.0 ns/op |
|-----|--------------|--------------|
| 1   | 772,306      | 742,362      |
| 2   | 774,390      | 739,652      |
| 3   | 778,905      | 738,775      |

**Result**: −4.5% improvement (not a regression). Allocations unchanged (123 allocs/op, ~10.8 KB/op). The additional `RevokeRefreshToken` call against the in-memory store adds no measurable overhead.
